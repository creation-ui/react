# renderSelection

This an example of the default selection rendering function. Based on that you can create your own `renderSelection` function.

By default T = [SelectOptionDefault](/docs/components/select#SelectOptionDefault)

```tsx
import { SelectOptionDefault } from './types'

const _renderSelection = (value: T) =>
  value ? getOptionLabel(value) : <Placeholder>{placeholder}</Placeholder>
```
