## States and variants

`SelectOption` can be used to list either a unique option selection or multiple at once. Depending on this the component will have a different UI presentation as indicated in the table below.

Component will always take up all available width as it has class `w-full`.

| State      | Single                                    | Multiple                                          |
| ---------- | ----------------------------------------- | ------------------------------------------------- |
| `default`  | <SelectOption option={option} />          | <SelectOption option={option} multiple />         |
| `active`   | <SelectOption option={option} active/>    | <SelectOption option={option} active multiple/>   |
| `selected` | <SelectOption option={option} selected /> | <SelectOption option={option} selected multiple/> |
| `disabled` | <SelectOption option={option} disabled /> | <SelectOption option={option} disabled multiple/> |
