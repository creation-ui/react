# Show

The Show component is a utility isnpired by SolidJS for conditional rendering with smooth transitions. It displays its children based on the truthiness of the `when` prop, and can display a `fallback` component if `when` is false.

In this example, we're using the `Show` component to conditionally render a "Log in" or "Log out" button based on whether the user is logged in.
Here's how it works:

```tsx
export const ShowExample = () => {
  const [loggedIn, setLoggedIn] = useState(false)

  const login = () => setLoggedIn(true)
  const logout = () => setLoggedIn(false)

  return (
    <Container variant='column'>
      <Show when={loggedIn} fallback={<Button onClick={login}>Log in</Button>}>
        <Button onClick={logout} variant={'outlined'}>
          Log out
        </Button>
      </Show>
    </Container>
  )
}
```

<ShowExample />

In the above code:

`when={loggedIn}`: This prop determines which content to display. If `loggedIn` is `true`, the `Show` component will display its `children`. If `loggedIn` is `false`, it will display the `fallback` component.
`fallback={<Button onClick={login}>Log in</Button>}`: This prop specifies what to display if `when` is false. In this case, it's a "Log in" button.
The children of the `Show` component `<Button onClick={logout} variant={'outlined'}> Log out</Button>`: This is what will be displayed if `when` is true. In this case, it's a "Log out" button.
The `Show` component uses animations from `framer-motion` to smoothly transition between the `children` and `fallback` components when the value of `when` changes.
