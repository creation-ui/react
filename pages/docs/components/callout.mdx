## Status

This component supports `status` prop with values `success`, `warning`, `error`, `active`.
Default style is applied when `status` prop is left not defined.

## Clearable

Close button can be added to the callout. It is visible when `onClose` callback is provided:

<Container variant='column'>
  <Callout
    status='error'
    title='Error'
    content='There was an error while processing your request. You can close this prompt now.'
    icon={<Icon path={mdiAlertCircleOutline} size={1} />}
    onClose={fn}
  />
</Container>
