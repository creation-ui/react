# Select

Select component is using Listbox component from
[headless ui](https://headlessui.com/react/listbox) library.

## Basic usage example

```js copy

const [selection, setSelection] = useState<Option | never>()

  return (
    <Select
      size='sm'
      options={options}
      label={'Select'}
      value={selection}
      onChange={setSelection}
    />
  )

```

## Selecting multiple options

In this example, we are using the `multiple` prop to allow selecting multiple options.

<br />
<SelectMultipleExample clearable clearableButtonText='Clear selection' />
<br />
Because of the usage of the Listbox component under the hood, the `value` and `onChange`
props change the signature:{' '}

| Property | `!multiple` | `multiple`    |
| -------- | ----------- | ------------- |
| value    | `T`         | `T[]`         |
| onChange | `T => void` | `T[] => void` |

For details see [properties table](#onChange)

## Error state

```js copy

const [selection, setSelection] = useState<Option | never>()

  return (
    <Select
      clearable
      clearableButtonText="Clear selection"
      options={options}
      label={'With error'}
      value={selection}
      onChange={setSelection}
      error={'You must select an option'}
    />
  )

```
