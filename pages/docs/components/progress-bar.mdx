## Formatting value

Value can be custom formatted with your own function that takes `value` and returns `ReactNode`.
`showValue` needs to be set to true for this to work.

```tsx copy
<ProgressBar
  value={65}
  formatDisplayValue={value => `Preparing for lunch: ${value}%`}
  showValue
/>
```

<Container>
  <ProgressBar
    value={65}
    formatDisplayValue={value => `Preparing for lunch: ${value}%`}
    showValue
  />
</Container>
