## What if I a color other than `status`?

For this use case, you can use the `style` prop to customize the color of the chip. For example:

<Container>
  <CustomChip />
</Container>

Be aware that this will override the default coloring styles of the chip. Notice that bg hover effect was lost as it came from now disabled `status` styling.

```tsx
export const CustomChip = () => (
  <Chip
    label={'Indefatigable'}
    style={{
      color: 'rgba(255, 153, 0)',
      backgroundColor: 'rgba(255, 153, 0, 0.1)',
      borderColor: 'rgba(255, 153, 0, 0.5)',
    }}
  />
)
```
